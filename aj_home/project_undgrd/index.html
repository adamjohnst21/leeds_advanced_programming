Nearest pub program

A python 2.7 program for identifying pubs within 30 minutes walking distance and ranking them relative to another. This was the deliverable for the Understanding GIS project of second semester, BSc Geography

Program description

This program is designed to identify pubs from OSM data, and calculate which are within a 30 minute walk from a given point along a network of roads. Walkable pubs are published on a map using the mapik library, with their shortest routes. Pubs are then ranked in suitability based on their overall distance from the starting point and then the after-pub destination (E.g. a train station).

Download folder includes the program (9466841.py), demonstration data and a project report.
