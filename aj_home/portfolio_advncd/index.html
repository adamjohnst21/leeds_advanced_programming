Advanced programming portfolio

Collection of work carried out as part of a course module. This content is part of the advanced-programming portfolio of second semester, MSc GIS

ArcPy practicals

Practicals 1 to 4 covered the use of python with ArcGIS. This included using model builder, scripts, addins and GUIs. Throughout, a similar spatial analysis example was followed, inolving the topological relationship of points to polygons, including explosion damage to buildings and likelihood of burglaries. The main output of these practicals is an Arc addin which runs the Trafford crime model, putting to practice all the skills from 1-4. This can be downloaded by clicking the main button below. Database and XML practicals

Practicals 5 & 6 covered databases, SQL and XML creation/styling in python. In practical 5, we used python to first create then read a database, with data following on with the burglary theme. SQL was used to retreive specific data. In practical 6 we looked at the different styling methods of XML (DTD and XSL), and how to use the etree library in python to parse, add to and transform XML.

Pandas and NLTK practicals

Practicals 7 & 8 covered data proccessing and language processing with the pandas and nltk libraries respectively. Practical 7 used jupyter notebook to work through online pandas and bokeh libraries tutorials. Pandas included accessing/analysing data, while bokeh focused on the display of data.
