Radar waterbodies classification program

A python program for classifying waterbodies using Sentinel-1 radar imagery. This was the deliverable for the core-programming project of first semester, MSc GIS

Program description

This program is designed to work with Sentinel-1 radar imagery, in 256-bit geotiff format. Taking advantage of the low reflectance of microwaves from water, water bodies can be classified, with an accuracy of ~80%. A raster mask is produced as a result, where 0=Land and 1=Waterbody. A reference image can be used to produce the accuracy of classification

Download folder includes the program (Main_program.py), demonstration imagery and a project report.
