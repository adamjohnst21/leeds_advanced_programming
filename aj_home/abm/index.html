Agent Based Model program

A python program is an ABM that creates agents that interact with their environment. This was the deliverable for the core-programming portfolio of first semester, MSc GIS

Program description

This program is designed to create a user specified number of Agents, which can move randomly, interact with other agents and consume their environment. This process is displayed in a GUI as a graph, which shows Agents changing in size and the environment decreasing in size. Outputs include the final image and a report.

Download folder includes the program (finalModel.py), ajoining framework (agentFramework.py) and environmental data (in.txt).
